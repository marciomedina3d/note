#### Lista de comandos básicos Linux

Version 1.0.0

------

Lembrete dos comandos principais do Linux

###### Super Usuário

```
sudo -i
```

###### Alterando a Senha

```
passwd
```

###### Sair do super usuário

```
exit
```

###### Instalando arquivos deb

```
sudo dpkg -i package_file.deb
```

###### Instalando arquivos deb

```
sudo apt-get remove
```

###### Criar pasta

```
mkdir exemplo
```

###### Listar Pasta

```
ls
```

###### Apagar Pasta

```
rm r- exemplo
```
